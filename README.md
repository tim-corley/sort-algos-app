# Algorithm Sorter

A React application that will sort a (json) data set by leveraging a selected sort algorithm.

Available sort algorithms:

- [ ] Regular Sort
- [ ] Bubble Sort
- [ ] Merge Sort
- [ ] Insertion Sort
- [ ] Quartile Sort

To Run:

```bash
$ npm install
$ npm run start
```
